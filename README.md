# 🐝 Vue Clean Architecture App

## 📦 Prerequisites

- Node.js v16.17.0
- Yarn v1.22.19

## 🏃 Running

Install node libraries

```sh
yarn install
```

Running in dev mode

```sh
yarn dev
```

Building for production

```sh
yarn build
```
